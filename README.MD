# StoneCraft Masonry – Multi-Page Business Website

Live Demo (GitHub Pages): https://YOUR-USERNAME.github.io/stonecraft-masonry/

A clean, modern, mobile-responsive website built for a local masonry business.  
This project is designed as a real client-style portfolio piece for freelance work.

## Pages
- Home
- About
- Services
- Gallery
- Contact

## Features
- Mobile-first responsive layout
- Sticky navigation + mobile menu (JavaScript)
- Clean UI components (cards, panels, CTA buttons)
- Gallery layout ready for real project photos
- Contact form UI (demo) – can be connected to Formspree/EmailJS/backend

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run Locally
1. Download/clone the repo
2. Open `index.html` in your browser

## Portfolio Notes
- Static version is ideal for fast performance and simple hosting.
- WordPress version can be created using the same layout with Gutenberg/Elementor.
